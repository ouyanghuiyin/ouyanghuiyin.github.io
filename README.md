# Huiyin Ouyang

This is the source repository for [Huiyin Ouyang's academic website](https://ouyanghuiyin.github.io/), built with the [AcademicPages](https://github.com/academicpages/academicpages.github.io) Jekyll theme and deployed via GitHub Pages.

The site is automatically built and deployed by the workflow in `.github/workflows/publish.yaml`.
